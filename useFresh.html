Use fresh.css

Using fresh is easy. Just link the css into the head, and the JavaScript and Jquery links on the bottom of the body.

<link rel = "stylesheet" type = "text/css" href = "https://cdn.jsdelivr.net/gh/HunterBall/fresh.css@latest/fresh.min.css">
<script src = "https://cdn.jsdelivr.net/gh/jquery/jquery@latest/dist/jquery.min.js"></script>
<script src = "https://cdn.jsdelivr.net/gh/HunterBall/fresh.css@latest/fresh.min.js"></script>

Fresh.css Components


Navigation

Fresh.css houses multiple different navigations. You can use one or multiple of these in order to create your site or web app.


Top Nav


Nav Title Item 1 Item 2 Item 3

Fresh has two top navigation styles: fixed and sticky. You can use either of these in order to create a functional top navigation.

<div class = "nav top"></div><!-- Fixed Top Nav -->

The default top navigation is fixed, and it doesn't matter where it is placed in the DOM.

<div class = "nav top sticky"></div><!-- Sticky Top Nav -->

The sticky navigation will stay within the DOM until it reaches the top, and should be placed accordingly.

Side Navigation

Side navigation, which also doubles as part of the mobile navigation, can be used as your default navighation or as a sub-navigation, when combined with a top navigation

<div class = "nav side"></div> <!-- Fixed Side Navigation -->

Nav Components

Creating great looking navigations has never been easier. Use item for any normal link, and title for the navigation title.

<div class = "nav"> <!-- Default Navigation (Same as top nav, without sticky or fixed position) -->
  <a class = "title">Nav Title</a> <!-- Nav Title -->
  <a class = "item">Item 1</a> <!-- Nav Item -->
  <a class = "item">Item 2</a>
  <a class = "item">Item 3</a>
</div>

On side navigations, you can also use the <hr> tag to divide the nav into sections.

Main Component

The main component is used to contain elements in relation to navigation and modals.

<div class = "main"></div><!--No Nav or Sticky Nav used-->
<div class = "main nav-top"></div><!--Top Nav used-->
<div class = "main nav-side"></div><!--Side Nav used-->
<div class = "main nav-side nav-top"></div><!--Side Nav and Top Nav used-->

Header

The header component can be used for most headers, and is easy to implement.

<div class = "header"></div>

Container Component

The container component is used for most items, to divide them into sections. This makes editing and design look good. The container also keeps the padding for all elements, instead of the main, so that items you want to span the entire main content area can and everything else keeps the normal padding.

<div class = "container"></div>

Columned Items

Any element can be placed into responsive columns. Each column distributes width evenly among each item.

<div class = "columns"><!--column parent element-->
  <div class = "column"><!--responsive column-->
    <!--Insert Whatever you want here-->
  </div>
</div>

Just add the column element for each column you want. Fresh will do the rest!

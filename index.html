What is fresh.css?

Fresh.css is a CSS and JavaScript framework for small project websites and lightweight web apps. Fresh uses old and modern techniques, mixed with unique functionality and features to give you the easiest experience possible, while still giving you the freedom and uniqueness your website deserves.

Easy to Set Up

Just link to the jsDelivr to get the minified css, JavaScript, and JQuery and you're good to go. It's that easy.

Easy to Customize

Using css variables, you can easily override colors, styles, and more to make Fresh unique for you and your project.

Deploy Fast

Don't worry about the fundamentals. We got that. Just work on making your web app, experience, or site amazing.

Fresh.css Features

Fresh.css isn't the one and only option, and neither is any other framework. Every framework has different features and different uses. However, if you're looking for a great framework, fresh.css may be exactly what you're looking for.

Use Fresh.css

If you want to use Fresh.css, just link the following stylesheet link into your html head:


<link rel="stylesheet" type="text/css" href="https://cdn.jsdelivr.net/gh/HunterBall/fresh.css@latest/fresh.min.css">


Then, at the end of your body tag, insert the following two lines of code:


<script src="https://cdn.jsdelivr.net/gh/jquery/jquery@latest/dist/jquery.min.js"></script>

<script src="https://cdn.jsdelivr.net/gh/HunterBall/fresh.css@latest/fresh.min.js"></script>

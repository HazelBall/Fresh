What is fresh.css?

Fresh.css is a CSS and JavaScript framework for small project websites and lightweight web apps. Fresh uses old and modern techniques, mixed with unique functionality and features to give you the easiest experience possible, while still giving you the freedom and uniqueness your website deserves.

Easy to Set Up

Just link to the jsDelivr to get the minified css, JavaScript, and JQuery and you're good to go. It's that easy.

Easy to Customize

Using css variables, you can easily override colors, styles, and more to make Fresh unique for you and your project.

Deploy Fast

Don't worry about the fundamentals. We got that. Just work on making your web app, experience, or site amazing.

Fresh.css Features

Fresh.css is what we call a mid-tier framework. Smaller, lightweight frameworks offer fast speeds but can only do so much for you, and larger, all-encompassing frameworks offer everything you could ever need, but they don't work well with custom code and other frameworks. Fresh offers a middle ground to this problem: a speedy framework with options and amazing features, that works well with any other CSS or JS framework.

Use Fresh.css

If you want to use Fresh.css, just link the following stylesheet link into your html head:

<link rel="stylesheet" type="text/css" href="https://cdn.jsdelivr.net/gh/HunterBall/fresh.css@1.0.0/fresh.min.css">

Then, at the end of your body tag, insert the following line of code:

<script src="https://cdn.jsdelivr.net/gh/HunterBall/fresh.css@1.0.0/fresh.min.js"></script>
